# Black Souls: Dark Tale 

Проект представляет собой игру в жанре Dark Fantasy, вдохновленную серией **Black Souls**.

### Описание сюжета
События разворачиваются после завершения первой части оригинальной игры. Сюжет предлагает альтернативный путь развития истории, предлагая уникальное авторское видение мира, отличное от каноничного продолжения серии. Основной акцент сделан на повествовании и атмосфере темного фэнтези.

### Текущий статус
Проект находится в стадии активной разработки. Ведется работа над сценарием, наполнением игровых сцен и реализацией ключевых механик взаимодействия.
